Netherlands Job Market Analysis using Adzuna API
Project Overview

This project uses the Adzuna Jobs API to collect and analyze job postings across the Netherlands.
The goal is to explore hiring trends, top job categories, top hiring companies, and the most active job locations using Python and data analytics techniques.

The project automatically:

connects to the Adzuna API
collects live job market data
cleans and transforms JSON data
exports results to CSV
generates charts for analysis
